# Команды проекта (Makefile)

| Команда | Описание |
|---------|----------|
| `make install` | Создаёт виртуальное окружение и устанавливает зависимости. |
| `make run` | Запускает одиночную игру в терминале. |
| `make server` | Запускает сетевой TCP‑сервер. |
| `make client` | Запускает клиент для подключения к серверу. |
| `make test` | Запускает тестовый набор `pytest`. |
| `make lint` | Проверка кода `flake8` и форматирование `black --check`. |
| `make docker-up` | Сборка и запуск Docker‑композ. |
| `make docker-down` | Остановка и очистка Docker‑сервисов. |
| `make clean` | Удаляет `__pycache__` и виртуальное окружение. |

Эти команды позволяют быстро разворачивать и проверять проект в разных режимах.
