# Бесконечные крестики‑нолики (Infinite Tic‑Tac‑Toe)

Текстовое приложение, реализующее бесконечное поле 3‑в‑ряд в терминале.
Игра идёт между человеком (`X`) и простым ИИ (`O`).

## Как запустить

```bash
chmod +x run.sh   # сделать скрипт исполняемым (один раз)
./run.sh          # запустить игру через модуль src.main
```

Или напрямую без оболочки:

```bash
python -m src.main
```

## Сетевая игра

Для многопользовательской игры запустите отдельный сервер и подключитесь к нему
клиентом. Сервер отвечает за общую доску, отслеживает ходы и рассылает
обновления всем подключённым клиентам.

### Запуск сервера
```bash
python -m server.server   # прослушивает 0.0.0.0:5555 по умолчанию
```
Если необходимо изменить адрес / порт, отредактируйте константы
`DEFAULT_HOST` и `DEFAULT_PORT` в файле `server/server.py`.

### Запуск клиента
```bash
python -m src.network_client
```
Клиент по умолчанию подключается к `127.0.0.1:5555`. При необходимости можно
изменить значения `host` и `port` в начале файла `src/network_client.py`.

**Важно:** в среде с ограниченным сетевым доступом (sandbox) запуск
сервер‑клиент может быть недоступен. На обычном компьютере эти команды
работают без ограничений.

## Управление

- **Стрелки** – перемещение курсора по полю.
- **Пробел** – поставить свой символ (`X`).
- После формирования тройки одинаковых символов (как у игрока, так и у ИИ)
  соответствующие клетки удаляются, а игрок получает очко.
- **Q** – выйти из игры.

## Требования

- Python 3.6+ (используется только стандартная библиотека).
- Модуль `curses` (входит в стандартную поставку Python).

## Структура проекта

- `infinite_ttt.py` – основной скрипт игры.
- `run.sh` – простой оболочечный скрипт‑обёртка для запуска.
- `README.md` – текущий файл.
